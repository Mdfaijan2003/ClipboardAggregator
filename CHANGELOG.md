# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-30

### Added
- Background Windows tray app
- Clipboard text capture (Unicode)
- Paste-all hotkey (`Ctrl + Shift + V`)
- Max history cap (12 items)
- Time-based expiry cleanup (1 hour)
- Tray controls: pause/resume, clear history, auto-start toggle, exit
- Sensitive content filtering (API keys, JWT tokens, passwords, card numbers)
- Auto-start on first launch
