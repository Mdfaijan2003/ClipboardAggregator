# Privacy Policy – Clipboard Aggregator

_Last updated: 2026-04-28_

Clipboard Aggregator is a local desktop utility for Windows.

## What the app does
- Monitors clipboard text changes on your device.
- Stores recent clipboard text items in memory.
- Lets you paste combined clipboard history via hotkey.

## Data collection
Clipboard Aggregator does **not** collect personal data by default.
- No account
- No cloud sync
- No telemetry by default
- No analytics by default

## Data storage
- Clipboard items are stored locally in app memory.
- Items are automatically removed after the configured expiry window.
- If persistent storage is added in future versions, it will be documented here.

## Data sharing
Clipboard Aggregator does **not** share clipboard data with third parties.

## Security notes
Clipboard data can contain sensitive information. Users should:
- Pause capture when handling secrets
- Clear history when needed
- Keep system malware protection enabled

## Future changes
If privacy behavior changes (e.g., optional crash reports/telemetry), this file will be updated before release.

## Contact
For privacy concerns, open a GitHub issue or contact the maintainer listed in the repository profile.
