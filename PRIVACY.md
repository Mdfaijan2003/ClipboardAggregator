# Privacy Policy – Clipboard Aggregator

_Last updated: 2026-04-30_

Clipboard Aggregator is a local desktop utility for Windows.

## What the app does
- Monitors clipboard text changes on your device.
- Stores recent clipboard text items in memory (up to 12 items).
- Lets you paste combined clipboard history via hotkey (`Ctrl + Shift + V`).

## Data collection
Clipboard Aggregator does **not** collect personal data.
- No account required
- No cloud sync
- No telemetry
- No analytics

## Data storage
- Clipboard items are stored locally in app memory only.
- Items are automatically removed after 1 hour.
- No data is ever written to disk.

## Data sharing
Clipboard Aggregator does **not** share clipboard data with any third parties.

## Security notes
Clipboard data can contain sensitive information. The app automatically filters out:
- API keys and tokens
- JWT tokens
- Passwords (pattern-matched)
- Card numbers

Users should additionally:
- Pause capture when handling secrets
- Clear history when needed
- Keep system malware protection enabled

## Future changes
If privacy behavior changes, this file will be updated before release.
